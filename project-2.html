Project Overview

The project fetches random landscape images from Unsplash's API, dynamically changing the webpage's background. This feature ensures a visually appealing and ever-changing backdrop. You can stay informed about the crypto market with real-time data for Dogecoin fetched from Coingecko API. The dashboard includes a live clock that updates every second

This project also uses the Geolocation API, the project fetches the user's location and uses it to obtain real-time weather data from the OpenWeatherMap API.
