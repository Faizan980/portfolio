Project Overview

The project utilizes the "Deck of Cards" API to dynamically generate and shuffle a deck, ensuring a randomized and fair distribution of cards for each game session.

Experience real-time card drawing functionality with a visually pleasing display of drawn cards. The interface is intuitive, allowing users to witness the unfolding game dynamically.

Keep track of scores effortlessly with a clear display of computer and player scores. The project implements a scoring system that updates in real-time as the game progresses.

The logic behind determining the winner of each round is meticulously implemented. The comparison of card values and the subsequent update of scores add strategic depth to the game.

Crafted using HTML, CSS, and JavaScript, this project showcases proficiency in front-end development. The integration of asynchronous fetch requests demonstrates the ability to interact with external APIs securely.

The project not only offers a functional game but also prioritizes user engagement. The dynamic updates, clear scoring, and visually appealing card displays contribute to an immersive and enjoyable user experience.
